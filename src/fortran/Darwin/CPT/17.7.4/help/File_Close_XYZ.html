121-123: Closing Input Files

Use the Close X (121), Y (122), or forecast(123) file option under the File menu item to close an input file without restoring CPT's default program settings (as occura with File ~ New). Closing an X or Y file will reset CPT so that results need to be recalculated. If a forecast file is closed only the forecasts will be reset.
