581: Station Markers

By default, values for station data are plotted on a map using a cross in the shape of the letter 'X'. Alternatively a star can be used (in the shape of an X with a horizontal line), or a filled circle. Sizes of the markers can be rescaled if desired by setting the scaling factor to a value other than 1.0. The default size of the marker is multiplied by the scaling factor, and so a scaling of 2.0 will double the size of the marker, for example.
