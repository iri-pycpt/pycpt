View

There are five analysis methods available in CPT. The first four can be used to construct a forecast model, whereas the fifth (PFV) is specifically use to verify a set of probabilistic forecasts. The five analysis methods are directly available only using the Expert mode, and are:
  • 611: Canonical Correlation Analysis (CCA)
  • 612: Principal Components Regression (PCR)
  • 613: Multiple Linear Regression (MLR)
  • 614: Raw General Circulation Model Output (GCM)
  • 621: Probabilistic Forecast Verification (PFV)
To apply a specific method select the appropriate menu item from View.

These menu items may not be always available depending upon what type of X input file is open.

If the Beginner mode is selected, the View menu items are:

  • 631: Forecast
  • 632: Diagnostics
  • 621: Verification The five analysis methods can be accessed from Options ~ Models.
