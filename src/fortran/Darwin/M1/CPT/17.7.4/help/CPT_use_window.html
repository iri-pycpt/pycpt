CPT Window

The CPT Window is opened in full screen mode once an analysis method has been selected, or if a project file is opened. There should be seven menu items:
  • File
  • Edit
  • Actions
  • Tools
  • Options
  • View
  • Help

The CPT Window requests information related to the data to be used to construct the model. The Window contains prompts to request the most basic settings, such as input files, and information about the training period, but additional settings are available from some of the menu items.
