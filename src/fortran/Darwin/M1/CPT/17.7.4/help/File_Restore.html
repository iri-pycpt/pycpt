Resetting CPT Defaults

In the Windows version of CPT, the user's default settings are stored in $USERPROFILE/App Data/Roaming/CPT/CPt.ini in Windows version 6 and later (Spanish and Portuguese versions of Windows may have equivalent translated directory). This file is deleted and replaced with the system version (typically installed in C:\CPT\) if the File ~ Restore Defaults menu item is selected.

In most cases the user should notice little difference since the current version of CPT does not provide options to modify these defaults. However, if the file has been modified or corrupted somehow, then it can be restored using this menu item.
