621: Probabilistic Forecast Verification (PFV)

From the Title Window the Probabilistic Forecast Verification (PFV) menu item will set the CPT Window for PFV. The PFV option will restrict the type of input files that can be used. Specifically, the X input file must contain forecast probabilities, and thus contain the header line cpt_ncat=3 (see Input Datasets for further details). This menu item is not available if a probabilistic X input file is open.
