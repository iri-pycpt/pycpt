Save Goodness Index

The optimization steps indicated on the Results Window can be saved to an output file. Selecting this menu item will result in a prompt for an output file. The goodness index results will be saved to this file only when Actions ~ Cross-validated or Actions ~ Retroactive is activated. The results are not automatically saved to the file again if CPT is reset. This Gooness Index output file cannot be set after results have been calculated.
